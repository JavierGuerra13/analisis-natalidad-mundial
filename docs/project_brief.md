# Proyecto: Análisis de la Caída de Natalidad Mundial

## 📋 Resumen Ejecutivo

Este proyecto tiene como objetivo analizar la tendencia de disminución de la natalidad a nivel mundial, identificar los países donde se presenta este fenómeno, y explorar los factores que podrían explicar esta situación.

## 🎯 Objetivos del Proyecto

### Objetivo General
Realizar un análisis de datos completo sobre las tasas de natalidad globales para entender patrones, tendencias y factores asociados a la caída demográfica.

### Objetivos Específicos

1. **Mapear la tendencia global**: Identificar qué países experimentan caída de natalidad y cuáles mantienen tasas estables o crecientes.

2. **Análisis temporal**: Examinar la evolución de las tasas de fertilidad desde el año 2000 hasta 2023.

3. **Identificar factores causales**: Explorar correlaciones entre la natalidad y variables:
   - Económicas (PIB per cápita, desempleo, costo de vivienda)
   - Sociales (educación femenina, participación laboral)
   - Políticas (licencias parentales, políticas de familia)
   - Culturales (urbanización, cambios sociales)

4. **Casos de estudio**: Analizar países outliers (aquellos que van contra la tendencia).

5. **Generar insights accionables**: Producir conclusiones que puedan ser compartidas públicamente y generar conversación informada.
