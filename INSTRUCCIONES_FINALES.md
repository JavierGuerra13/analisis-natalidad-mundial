## 🚀 Próximos Pasos (en orden):

### 1. **Descargar el proyecto**
   - Descarga la carpeta `natalidad-mundial` completa
   - Guárdala donde quieras trabajar (ej: Documentos/proyectos/)

### 2. **Abrir en VS Code**
   ```bash
   cd ruta/donde/guardaste/natalidad-mundial
   code .
   ```

### 3. **Configurar Python** (si no lo tienes)
   - Descarga Python 3.10+ desde python.org
   - Durante instalación: marca "Add Python to PATH"

### 4. **Crear entorno virtual**
   ```bash
   python -m venv venv
   venv\Scripts\activate  # Windows
   # o
   source venv/bin/activate  # Mac/Linux
   ```

### 5. **Instalar dependencias**
   ```bash
   pip install -r requirements.txt
   ```

### 6. **Leer la documentación**
   - Empieza por `QUICK_START.md`
   - Luego lee los docs en carpeta `docs/`

### 7. **Empezar con los notebooks**
   ```bash
   jupyter notebook
   ```
   - Abre `notebooks/01_data_collection.ipynb`
   - ¡Empezamos a programar juntos!
